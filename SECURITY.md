| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within Vitals, please do **not** create a public GitHub issue.

### Process

1. **Report**: You send a detailed report of the vulnerability.
2. **Acknowledge**: We will acknowledge your report within 48 hours.
3. **Investigate**: We will investigate the issue and determine its impact.
4. **Fix**: We will work on a patch to resolve the vulnerability.
5. **Release**: We will release a security update and notify you.

We appreciate your efforts to responsibly disclose your findings and will make every effort to acknowledge your contributions.
